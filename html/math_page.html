Math Skills

This game is intended to help teach and reinforce the basic ideas of Mathematics.

Addition

This game uses addition when adding together the total damage done each round, and users are expected to be able to add together the total costs of units, refreshing, and leveling up in order to plan out a turn

Subtraction

This game uses subtraction when taking away the number of coins used for costs of units, refreshing, remaining health, and leveling up users are expected to use this in order to calculate if they will win and will be able to grasp an understanding by seeing the numbers and bars go down

Multiplication

This game uses multiplacation when dealing damage of type advantages. Users are expected to be able to use this to predict who will win in a fight

Division

This game uses division when choosing a shield. The user is expected to be able to chose a divisor of the incoming damage in order to prevent damage
